# WallCl

WallCl은 2020 SRC/IRC 지능형 창작 로봇 대회에서 제작한 벽면 등반 로봇입니다. 건물 붕괴와 같은 재난 환경에서 벽에 붙어 이동하며 구조 물품을 전달하는 상황을 목표로 설계했습니다. 게코 도마뱀의 벽면 이동 방식에서 아이디어를 얻어, 전자석으로 벽면에 고정하고 서보모터로 다리를 움직이는 구조를 구현했습니다.

프로젝트 정리 페이지: https://hyunjem.notion.site/wallcl

## 프로젝트 개요

로봇은 다리와 연결된 서보모터를 일정 각도 순서로 회전시켜 전진과 좌우 방향 전환을 수행합니다. 이동 중에는 전자석을 이용해 벽면에 부착된 상태를 유지하고, 초음파 센서로 전방 장애물을 감지해 이동 방향을 판단하도록 설계했습니다.

기구 설계는 가벼운 무게와 높은 강도를 동시에 만족하는 데 집중했습니다. 본체는 PLA 필라멘트로 출력하고, 안정성을 높이기 위해 삼각형 구조를 적용했습니다. 다리는 종이를 T자 형태로 접었을 때 강성이 높아지는 원리에서 착안해 T자 구조로 설계했습니다.

## 사용 부품

| 구분 | 부품 |
| --- | --- |
| MCU | Arduino Uno (ATmega328) |
| 서보모터 | MG996R |
| 서보 드라이버 | PCA9685 PWM Servo Driver |
| 초음파 센서 | HC-SR04P |
| 전자석 | WF-P30 |
| 전자석 구동 | IRF520 MOSFET |
| 본체/다리 | PLA 3D 프린팅 구조물 |
| MCU 전원 | 9V 배터리 2개 병렬 연결 |
| 서보/센서 전원 | 5V Linear Regulator 출력 |
| 전자석 전원 | 외부 21V 전원 |

## 핀맵

| 구분 | 연결 |
| --- | --- |
| PCA9685 SDA/SCL | Arduino Uno I2C |
| 전자석 0 | D3 |
| 전자석 1 | D4 |
| 전자석 2 | D5 |
| 서보 채널 0 | PCA9685 0번 채널 |
| 서보 채널 1 | PCA9685 1번 채널 |
| 서보 채널 2 | PCA9685 2번 채널 |

초음파 센서는 전방 장애물 감지용으로 설계에 포함되며, 다리 이동 반경을 고려해 약 2cm 여유 거리를 두고 장애물을 판단합니다.

## 시스템 동작 흐름

1. Arduino가 PCA9685 서보 드라이버를 50Hz로 초기화합니다.
2. D3, D4, D5 전자석 제어 핀을 출력으로 설정합니다.
3. 기본 상태에서 모든 전자석을 켜 벽면 고정력을 유지합니다.
4. Serial 명령을 입력받아 이동 방향을 선택합니다.
5. 명령 `0`은 전진, `1`은 좌회전, `2`는 우회전에 대응합니다.
6. 선택된 방향에 따라 두 개의 서보 채널을 한 쌍으로 움직입니다.
7. 사전에 계산한 각도 시퀀스를 PWM 펄스로 변환해 서보모터에 전달합니다.
8. 동작이 끝나면 두 서보를 기준 위치로 복귀시킵니다.

## 주요 기능

- 전자석 기반 벽면 부착 유지
- PCA9685 기반 다중 서보모터 제어
- Serial 명령 기반 전진/좌회전/우회전 동작
- 각도 시퀀스를 이용한 다리 보행 패턴 구현
- 초음파 센서 기반 전방 장애물 감지 설계
- 3D 프린팅 본체와 T자형 다리 구조

## 실행 방법

1. Arduino IDE에서 `src/code.ino`를 엽니다.
2. `Adafruit PWM Servo Driver Library`를 설치합니다.
3. Arduino Uno와 PCA9685, 전자석 제어 회로를 핀맵에 맞게 연결합니다.
4. 보드와 포트를 선택한 뒤 업로드합니다.
5. Serial Monitor 또는 외부 제어 프로그램에서 명령을 전송합니다.

| 명령 | 동작 |
| --- | --- |
| `0` | 전진 |
| `1` | 좌회전 |
| `2` | 우회전 |

## 시연 결과 또는 기대 동작

- 전자석이 활성화되어 로봇이 벽면에 부착됩니다.
- 명령 `0` 입력 시 서보 채널 0, 1이 전진 보행 패턴으로 움직입니다.
- 명령 `1` 입력 시 서보 채널 2, 0이 좌회전 패턴으로 움직입니다.
- 명령 `2` 입력 시 서보 채널 1, 2가 우회전 패턴으로 움직입니다.
- 보행 동작 후 서보는 기준 위치로 복귀하고 다음 명령을 기다립니다.

## 설계 포인트

- 전자석의 자력이 로봇 무게를 견딜 수 있도록 본체를 경량화했습니다.
- 삼각형 프레임으로 벽면 부착 상태에서 구조 안정성을 높였습니다.
- 다리는 T자 형태로 설계해 낮은 무게로 강성을 확보했습니다.
- 이동 정확도와 1회 이동 거리를 고려해 다리 길이와 전자석 부착 위치를 조정했습니다.
- 서보모터 각도는 로봇 팔 사이 거리와 가동 범위를 계산해 시퀀스로 구성했습니다.

## 개선 가능성

- 초음파 센서 입력을 현재 이동 명령과 통합해 자동 장애물 회피 루틴 구현
- 전자석 ON/OFF 타이밍을 보행 단계별로 분리해 소비 전력 절감
- 이동 시퀀스를 테이블화해 속도와 보폭을 쉽게 조정
- 배터리 전압 모니터링 추가
- 기구 모델과 코드의 서보 채널 매핑 이미지 추가
